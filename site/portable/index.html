Portable memory

Take your project memory into any web AI chat

Neonmem lets you carry your memory with you — into ChatGPT or Claude on the web — so the AI there answers from your project, not its best guess. Then you carry the updates back.

What is it?

New

Your project memory normally lives in the Neonmem app on your machine. Portable memory lets you export that memory as a single file and drop it straight into a web AI chat — so even when you're away from the app, on someone else's computer, or your coding agent has run out for the day, you still have a grounded, knowledgeable assistant.

Why you'd want it

  • One memory, everywhere. The same project knowledge in the app and in your web chats.
  • Works on the web — even free chats. No install on the machine you're using.
  • Grounded answers. The web AI answers from your docs and decisions, or says it doesn't know — instead of inventing.
  • Your updates come home. Remember something in the chat, and carry it back into your cartridge for next time.

How it works — three simple moves

  1. Bundle. In the Neonmem app, export your project as a portable memory bundle — one file.
  2. Use it on the web. Drop that file into your ChatGPT or Claude chat and ask about your project. The AI reads your memory and answers from it — grounded, no install.
  3. Carry it back. Anything new you decide in the chat can be saved back into the memory, and brought home to your cartridge — so nothing is lost.

Think of it like a memory stick for your project: insert it into a chat, work, take it back out with your changes.

The same cartridge

It's one memory — the cartridge you build in the app is the same one you carry to the web. The desktop app is the full experience (it remembers automatically as you work); the portable bundle is how you take that memory with you when you're not at the app. Learn how the app works →
