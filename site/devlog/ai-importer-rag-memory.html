Developer Corner

An offline AI importer that builds a full RAG memory

The Neonmem team · Jun 19, 2026 · ← Back to Dev Corner

Neonmem has always grown a smart memory as you work — a reasoning graph with tiers, plans and a self-built comprehension layer. This release adds the other half: a way to pour everything you already have into that memory, run by an AI that lives entirely on your machine.

A full RAG memory — on top of the smart one

Under the reasoning graph, every import now lands in a new facts layer: a canonical, deduplicated, vector-searchable knowledge base. It's retrieval-augmented memory done properly:

  • Deduplication. The same fact stated three different ways across three documents becomes one canonical fact — with all three sources remembered.
  • Hybrid retrieval. Vector similarity and keyword matching, so the right fact surfaces whether you remember the wording or just the meaning.
  • It forgets, on purpose. Facts nothing references any more quietly fade and are swept — a biologically-inspired garbage collector that runs while the memory "sleeps", so the cartridge stays sharp instead of bloating.
  • Always rebuildable. The facts are an index over the original source, which the cartridge keeps — so if the model improves, the whole knowledge base can be regenerated from the ground truth. Nothing is ever truly lost.

One smart cartridge

Here's the part we're most excited about. The importer keeps the full source corpus inside the cartridge — content-addressed and compressed — as the ground truth. Gather your Claude history, a pile of PDFs, internal docs and a few Teams transcripts, import them, and you get one smart file that holds it all: the compressed originals and the searchable knowledge built from them.

So you not have to reference originals each time all in one place now.

And because that file can now hold your whole corpus, it supports opt-in encryption at rest (AES-256-GCM) — a private vault, not a folder of loose documents.

Bring your Claude history in one click

The importer finds your Claude Code conversations automatically — newest first, just like --resume — so you never have to go hunting for log files. It strips the tool-call and system noise and imports the actual conversation as knowledge.

How it stays small and fast

A document set that's hundreds of megabytes on disk becomes a cartridge a fraction of the size: duplicate facts collapse, source chunks are stored once and compressed, and the vector index uses 1-bit quantization — up to 32× smaller than raw vectors, with the recall preserved. It all persists as compact binary, never bulky text.

Honest about what's under the hood

The importer is a compact, self-contained AI engine that runs entirely on your machine — no cloud, no API key, no GPU. It's built on proven, permissively-licensed open foundations, and everything ships as a single binary. The full third-party attributions travel with every download.
