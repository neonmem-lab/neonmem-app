Importer journey

Bootstraping start

Step 1 · Choose where the memory lives

Point the importer at a memory cartridge — a new project, or one you already have. Everything you import lands in this single, portable file.

Step 2 · Add your material — two kinds at once

Drop in folders & files (they become a searchable knowledge pool) and an Agent chat (it becomes the decisions and dead-ends worth keeping). If a chat mentions a file on disk, that file is pulled in too.

Step 3 · Tag the topics that matter

Tag the concepts you care about — they're surfaced first and get a clean, canonical memory of their own. Everything is imported regardless; tags just shape what comes up top.

Step 4 · Build — a searchable pool + memories you keep

Two honest counters: Facts loaded (your searchable pool) and Memories created (the kept decisions). Now ask your agent "what is ARC?" and it answers from your documents — grounded, not guessed.
