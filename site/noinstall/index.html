No-install

Run the app without installing

Same Neonmem app, zero installation. Unzip it to a folder or a USB stick and run it.

Why no-install?

  • No installer. CVL to system folders or the registry.
  • Runs from anywhere. A project folder, an external drive, a USB stick.
  • Clean. Want it gone? Delete the folder. That's it.
  • Includes the command-line tool and a short README, right in the folder.

Note: this is about running the app without installing. To take your memory into a web AI chat, see Portable memory.

How to use it

Windows (no-install)

  1. Download neonmem-0.9.8-portable.tar.gz.
  2. Extract it (Windows can open .tar.gz directly, or use 7-Zip) into any folder — or onto a USB stick.
  3. Open the folder and run the Neonmem executable. No installer.
  4. For the terminal tool, run neonmem --help from the same folder — the included README lists the commands.

Linux (AppImage)

  1. Download the AppImage.
  2. chmod +x Neonmem-0.9.8-x86_64.AppImage
  3. ./Neonmem-0.9.8-x86_64.AppImage — runs as-is, no root.
